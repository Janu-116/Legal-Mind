# Legal Mind AI

Legal Mind AI is an AI-powered legal assistant designed to help users understand legal concepts, analyze legal documents, and obtain legal information through a modern web interface.

## Features

* AI-powered legal assistance
* Legal query resolution
* Legal document analysis
* Responsive user interface
* Fast AI responses using Llama 3.3
* Scalable backend architecture

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js
* Groq SDK

### AI Model

* Llama 3.3 70B Versatile

### Database

* PostgreSQL (in progress)

### DevOps

* GitHub Actions
* Vercel
* Render
